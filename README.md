### 💡 Troubleshooting

- **Installation fails:** Run as Administrator/sudo and temporarily disable any antivirus software.
- **Simulator won't launch:** Ensure your graphics drivers are up to date and you have the latest VC++ redistributables.
- **Performance issues:** Close other background applications. This tool is optimized but benefits from free system resources.
- **Cards not appearing:** Use the "Refresh Library" button within the tool's settings menu.
